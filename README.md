# Sentiment Analysis using NLP

## Objective
Build a sentiment analysis model using NLP preprocessing and ML models.

## Dataset
IMDb Movie Reviews Dataset

## Steps Performed
- Data Cleaning & Preprocessing
- TF-IDF Feature Extraction
- Model Training (Logistic Regression, Naive Bayes, Decision Tree)
- Model Evaluation

## Results
- Logistic Regression: 88.6% Accuracy
- Naive Bayes: 85.1% Accuracy
- Decision Tree: 71.8% Accuracy

## Conclusion
Logistic Regression performed best for this task.
